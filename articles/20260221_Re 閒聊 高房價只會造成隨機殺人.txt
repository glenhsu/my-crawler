: 很遺憾，是這樣沒錯
: 重點不是高房價，而是貧富差距的問題
: 貧富差距會影響人的心理狀況
: 究其原因，是因為「社會評價威脅」，這是人們最大的壓力來源
: 而什麼叫「社會評價威脅」?
: 依照阿德勒的講法:
: 「生而為人，意味著感受自卑。」
: 也可以解讀成:
: 「生而為人，意味著對他人的鄙夷眼光高度敏感。」
: 簡單來說，別人怎麼看你，很重要
: 如果在社會階級中取得較高地位
: 那就隱含著自己比別人更強、更優秀，這會帶來自豪、自信，並免於自我懷疑
: 相反的，如果社會位階較低，感受到輕賤與侮辱時
: 這種屈辱感往往會誘發皮質醇上升，讓人感受到焦慮與憂鬱
: 在過去貧富差距沒那麼明顯的年代，社會位階的落差還沒那麼大
: 人們還視彼此為平等的個體
: 但隨著經濟發展
: 人們越來越看重社會地位，也越來越在乎社會評價
: 所以有沒有發現，之前的公車司機被學生嘴個幾句就暴走?
: 原因就在這
: 那種恥辱的感覺，就是根基於想像他人看待自己的社會評價反饋
: 因為在大眾，或至少在被揍的那位學生跟司機的心中
: 公車司機就不是個「體面」的職業
: 而那位司機心裡就是對此感到很自卑
: 是以產生了後續的衝突
: 依照心理學家James Gilligan的說法
: 「暴力行為是一種手段，旨在躲避或消除羞愧或屈辱感。」
: 透過與犯人的交談，他發現暴力的導火線總是與尊嚴的威脅有關
: 也就是讓人感到受辱的行為，然而觸發暴力的起始事件往往無足輕重
: 但人們從小到大或多或少都曾經受過貶低
: 大多數的人都不會以暴力來回應
: 因為他們有別的方式建立自尊自信
: 比如說好的工作、收入、房子、車子
: 有親近友善的朋友、家人，有值得驕傲的學經歷，有能帶來地位與前途的技能
: 但少數人就不是啊
: 他們(尤其是底層的年輕男性)就是無法取得良好社會地位該有的各種標準配備
: 又必須想辦法維持面子跟僅有的一點點地位
: 因此往往在自尊受損時牙起來，猛然爆發激烈的反應
: 是以，在貧富差距越懸殊的社會
: 就越多人缺乏維護自信自尊的保護與緩衝
: 當社會階級越僵固，那人們對難堪與羞辱就會越敏感
: 社會地位就變得越重要、競爭也越演越烈
: 因為有越來越多人被剝奪了「成功」的象徵
: 最後，當你僅有的自尊只剩下一點點而已，又被人踐踏時
: 或許反應就不再只是把對方的揍到鼻骨骨折而已。
: SO...
: 從某個角度來看，現代人真的是自我意識過剩
: 滿腦子都在想「別人怎麼看我」
: 擔心社會評價不夠高，努力地維持試圖呈現給別人的形象
: 這樣的脆弱，已然成為現代人心理狀態一部份
: 並且成為消費主義的養分
: 這也呼應前面KUSO大的文
: 「相對剝奪感」
: 非得要買房子、住月子中心、買當季旗艦機才滿意
: 其實那些訴求早已偏離主要功能
: 說到底
: 只是想展示自己過得「好不好」，維持社會地位形象
: 以及不想被看輕的指標罷了。
